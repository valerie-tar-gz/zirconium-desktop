# Zirconium Desktop
## What is this?
This is my personal Fedora Niri spin. It's heavily WIP, and **will** change as time goes on. I plan on implementing the Bazzite Kernel, HandHeldDaemon and other niche tools for my own use.

Again, this is a personal image. If you're after a fedora-bootc image with Niri to build off of use https://github.com/valerie-tar-gz/fedora-bootc-niri (Also WIP, but far less likely to have breaking changes)

## How to use
Either rebase using `bootc switch ghcr.io/valerie-tar-gz/zirconium-desktop:latest` or do a clean install using the ISO.
